# Генератор протоколов ГЭК

Веб-приложение для автоматизированного формирования протоколов и сопутствующих
документов Государственной экзаменационной комиссии (ГЭК).

## О проекте

Приложение предназначено для автоматизации работы секретаря ГЭК в образовательной
организации. Оно позволяет:

- загружать списки студентов из Excel-файлов (`.xlsx`, `.xls`) с автоматическим
  распознаванием структуры таблицы и извлечением метаданных (номер ГЭК, специальность);
- редактировать данные сессии (номер, даты, форму обучения), состав комиссии
  и список студентов;
- сохранять сессии в базе данных и открывать их повторно;
- генерировать готовые документы и выгружать их одним нажатием.

Решение избавляет от ручного дублирования данных, снижает количество ошибок
за счёт валидации и обеспечивает единообразие выходных документов.

## Технологический стек

**Backend:**
- Python 3.11+
- FastAPI — REST API
- Pydantic — валидация данных
- SQLAlchemy — ORM
- openpyxl / xlrd — чтение Excel-файлов

**Frontend:**
- React + TypeScript
- Vite — сборщик
- React Router — маршрутизация

## Требования

- Python 3.11 или выше
- Node.js 18 или выше
- npm (или yarn / pnpm)

## Запуск бэкенда

1. Перейдите в каталог бэкенда:

   ```bash
   cd backend
2. Создайте и активируйте виртуальное окружение:
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # Linux / macOS
   source venv/bin/activate

3. Установите зависимости:
   pip install -r requirements.txt
4. Запустите сервер разработки:
   uvicorn app.main:app --reload --port 8000

Запуск фронтенда

    В новом терминале перейдите в каталог фронтенда:
    bash
    cd frontend
Установите зависимости:
   npm install
Запустите сервер разработки:
   npm run dev
